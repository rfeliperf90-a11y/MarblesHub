MarblesHub

Gerenciador de Participantes para Marbles

Online

O que o MarblesHub faz:

MarblesHub é um bot Discord que facilita o gerenciamento de participantes para sorteios de Marbles. Integrado com o Marbles on Steam, permite que streamers exportem listas de participantes para gerar sorteios de forma rápida e divertida.

Principais Recursos

  • Inscrição rápida de participantes
  • Sistema de participação baseado em cargos
  • Exportação em CSV para Marbles on Steam
  • Gerenciamento de participantes
  • Controle de blacklist
  • Interface intuitiva e fácil de usar

Como Funciona

Configure o sorteio, deixe os participantes se inscreverem, gerencie a lista e exporte. Depois é só importar no Marbles on Steam e começar o sorteio quando quiser!

Comandos Disponíveis

/setup_sorteio
Configure um novo sorteio e defina os parâmetros
Admin
▼
Configurar Sorteio
Cria um novo sorteio com mensagem de inscrição e imagem/vídeo opcional
Definir Hashtag Obrigatória
Define uma hashtag obrigatória que participantes devem usar
Configurar Termos de Participação
Redefine os termos que participantes devem aceitar
Gerenciar Cargos Bônus
Adiciona/remove cargos que ganham mais fichas no sorteio
/gerenciar
Gerencia inscrições, participantes e blacklist
Admin
▼
Gerenciar Inscrições
Abre/fecha inscrições e controla alterações de dados dos participantes
Gerenciar Participantes
Lista, remove ou adiciona participantes manualmente
Gerenciar Fichas
Adiciona/remove fichas bônus de cargos e sincroniza dados
Controles do Sistema
Backup, restauração, resetar sorteio, adicionar moderadores
Exportar Lista
Exporta participantes em formato compatível com Marbles on Steam
Gerenciar Blacklist
Bloqueia/desbloqueia usuários e controla quem pode participar
/tag_manual
Concede fichas manuais (TAG) a um usuário específico
Admin
▼
Conceder TAG Manual
Concede fichas manuais (TAG) a um usuário específico (padrão: 1 ficha)
Quantidade Customizável
Define quantas fichas conceder (1, 2, 3, etc.). Para remover a ficha, basta colocar 0
Ficha Manual
Devido aos limites do Discord, este comando é manual pois ainda não há API de integração. Se disponibilizada posteriormente, esta função se tornará automática
/anunciar
Envia anúncios para canais
Admin
▼
Canal Destino
Escolha o canal onde o anúncio será enviado
Mensagem Personalizada
Escreva a mensagem do anúncio (use \n para quebra de linha)
Opções Avançadas
Use embed, adicione mídia (foto/vídeo) e botão de Termos de Participação
/reagir
Adiciona reações em mensagens automaticamente
Admin
▼
ID da Mensagem
Forneça o ID da mensagem que deseja adicionar/remover reações
Modo Adicionar/Remover
Escolha se quer adicionar (True) ou remover (False) as reações
Lista de Emojis
Digite os emojis desejados separados por espaço (ex: 1️⃣ 2️⃣ 3️⃣)
/bypass

Como Usar

Passo a passo:

1️⃣ Configure o sorteio com /setup_sorteio e defina as opções
2️⃣ Deixe participantes se inscreverem usando o botão de inscrição
3️⃣ Use /gerenciar para revisar e ajustar a lista
4️⃣ Exporte através da opção de exportação em /gerenciar
5️⃣ Importe no Marbles on Steam para gerar o sorteio

O arquivo exportado já está no formato que o Marbles on Steam aceita!
